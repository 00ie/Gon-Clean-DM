## Sobre

**Gon Clean DM** é uma ferramenta avançada para backup, limpeza/deleção e gestão profissional de suas DMs e grupos do Discord.

Principais funcionalidades:

- Backup completo de mensagens (TXT e CSV)
- Deleção massiva ou filtrada de mensagens (por data, termo, tipo de conteúdo)
- Seleção múltipla de canais
- Interface amigável, dark, com feedback visual.
- Exibição e busca rápida nas mensagens do canal
- Aba sobre com informações do usuário.

---

## Recursos

- **Backup multiplo:** Exporte chats, grupos, e DMs em .TXT ou .CSV
- **Deleção filtrada:** Elimine mensagens por range de data, texto ou tipo de conteúdo
- **Busca instantânea:** Localize mensagens em qualquer DM ou grupo rapidamente
- **Sobre profissional:** Tela com detalhes da sua conta logada, avatar e créditos ao desenvolvedor

## Instalação

### Requisitos

- Python 3.8 ou superior
- [pip](https://pypi.org/project/pip/)

### Instalação Rápida

Clone o repositório
git clone https://github.com/methzzy/gon-clean-dm.git
cd gon-clean-dm

Instale as dependências
pip install -r requirements.txt
